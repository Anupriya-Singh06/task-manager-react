# 📋 Task Manager App 🚀

A modern **Task Manager Web Application** built using **React.js** that helps users organize tasks efficiently with features like priority levels, due dates, filtering, progress tracking, and dark mode.

This project demonstrates **React fundamentals, component architecture, state management, and UI design**, making it a great **portfolio project for beginner developers**.

---

# ✨ Features

- ✅ Add tasks with **priority levels**
- 📅 Set **due dates**
- 🔍 **Search tasks**
- 🎯 **Task filtering** (All / Completed / Pending)
- ✔ Mark tasks as **completed**
- ✏ **Edit tasks**
- ❌ **Delete tasks**
- 📊 **Progress bar** showing completion status
- 🌙 **Dark mode toggle**
- 💾 **LocalStorage persistence**
- ⌨ Press **Enter key to add tasks**
- 🚫 **Empty state UI** when no tasks exist

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/task-manager-react.git
```

Navigate to the project folder:

```bash
cd task-manager-react
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm start
```

Open in browser:

```
http://localhost:3000
```

---

# 📦 Available Scripts
